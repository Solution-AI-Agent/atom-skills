# atom-skills

Atom 사내 Claude Code Skills 저장소

## 설치

```bash
git clone <repo-url> ~/.claude/plugins/atom-skills
```

## 구조

```
skills/
├── dev/        # 개발
├── hr/         # 인사
├── finance/    # 재무
├── sales/      # 영업
└── common/     # 공통
```

## 사용법

skill은 Claude Code에서 자동으로 인식됩니다.

```
/skill-name
```

## 기여

새 skill을 추가하려면 [CONTRIBUTING.md](docs/CONTRIBUTING.md)를 참고하세요.
