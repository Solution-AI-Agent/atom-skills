# 세율표 및 세액계산 기준 [ref: p.94, p.162-163]

## 개요
- 총급여액에서 근로소득공제를 차감하여 근로소득금액을 산출하고, 인적공제 등 소득공제를 적용하여 과세표준을 구한 뒤, 기본세율을 적용하여 산출세액을 계산한다.

## 산출세액 계산 흐름 [p.94, p.162]
```
총급여액 (= 연간근로소득 - 비과세소득)
  → (-) 근로소득공제
  → 근로소득금액
  → (-) 인적공제 (기본공제 + 추가공제)
  → (-) 연금보험료공제
  → (-) 특별소득공제
  → (-) 그 밖의 소득공제
  → (+) 소득세 소득공제 한도초과액
  → 과세표준
  → (x) 기본세율 (소득세법 제55조)
  → 종합소득산출세액
  → (-) 근로소득세액공제
  → (-) 기타 세액감면/공제
  → 결정세액
```

## 1. 근로소득공제 (소법 제47조) [p.94]
해당 과세기간의 총급여액에서 근로소득공제를 차감하여 근로소득금액을 계산한다. [p.94]

| 총급여액 구간 | 공제액 |
|---|---|
| 500만원 이하 | 총급여액의 70% |
| 500만원 초과 ~ 1,500만원 이하 | 350만원 + (500만원 초과분의 40%) |
| 1,500만원 초과 ~ 4,500만원 이하 | 750만원 + (1,500만원 초과분의 15%) |
| 4,500만원 초과 ~ 1억원 이하 | 1,200만원 + (4,500만원 초과분의 5%) |
| 1억원 초과 | 1,475만원 + (1억원 초과분의 2%) |

- **공제한도**: 2,000만원 [p.94]
- 과세기간이 1년 미만이거나 근로기간이 1년 미만인 경우 월할 계산하지 않고 전액 공제 [p.94]
- 총급여액이 근로소득공제에 미달하는 경우 총급여액을 공제액으로 한다 [p.94]
- 일용근로자가 아닌 사람이 2인 이상으로부터 근로소득을 받는 경우 합계액을 총급여액으로 하여 공제 [p.94]
- 총급여액은 비과세소득을 포함하지 않음 [p.94]

### 계산 사례 [p.94]
총급여액 3,380만원인 경우:
- 근로소득공제: 750만원 + (3,380만원 - 1,500만원) x 15% = 1,032만원
- 근로소득금액: 3,380만원 - 1,032만원 = 2,348만원

## 2. 종합소득세 기본세율 (소법 제55조) [p.83, 부록]
연말정산 시 과세표준에 기본세율을 적용하여 종합소득산출세액을 계산한다. [p.83]

| 과세표준 구간 | 세율 | 누진공제액 |
|---|---|---|
| 1,400만원 이하 | 6% | - |
| 1,400만원 초과 ~ 5,000만원 이하 | 15% | 126만원 |
| 5,000만원 초과 ~ 8,800만원 이하 | 24% | 576만원 |
| 8,800만원 초과 ~ 1억 5천만원 이하 | 35% | 1,544만원 |
| 1억 5천만원 초과 ~ 3억원 이하 | 38% | 1,994만원 |
| 3억원 초과 ~ 5억원 이하 | 40% | 2,594만원 |
| 5억원 초과 ~ 10억원 이하 | 42% | 3,594만원 |
| 10억원 초과 | 45% | 6,594만원 |

- 산출세액 = 과세표준 x 세율 - 누진공제액
- 부록 사례: 과세표준 14백만원 이하인 경우 기본세율 6% 적용 [부록 p.341]

## 3. 근로소득세액공제 (소법 제59조) [p.162-163]
근로소득이 있는 거주자에 대해서는 그 근로소득에 대한 종합소득 산출세액에서 근로소득세액공제 금액을 차감한다. [p.162]

### 가. 세액공제 금액 [p.162]

| 산출세액 구간 | 공제금액 |
|---|---|
| 130만원 이하 | 산출세액의 55% |
| 130만원 초과 | 71만5천원 + (130만원 초과분의 30%) |

### 나. 세액공제 금액 한도 [p.162]

| 총급여액 구간 | 세액공제 한도 |
|---|---|
| 3,300만원 이하 | 74만원 |
| 3,300만원 초과 ~ 7,000만원 이하 | 74만원 - [(총급여액 - 3,300만원) x 0.008]. 단, 66만원보다 적은 경우 66만원 |
| 7,000만원 초과 ~ 1억 2천만원 이하 | 66만원 - [(총급여액 - 7,000만원) x 1/2]. 단, 50만원보다 적은 경우 50만원 |
| 1억 2천만원 초과 | 50만원 - [(총급여액 - 1억 2천만원) x 1/2]. 단, 20만원보다 적은 경우 20만원 |

- 조특법 제30조의 중소기업 취업자에 대한 소득세 감면이 있는 경우: [p.162]
  - 세액공제액 = 소득세법 제59조 제1항에 따른 근로소득세액공제액 x (1 - 감면비율)
  - 감면비율 = 중소기업 취업자에 대한 소득세 감면액 / 산출세액

### 계산 사례 [p.162-163]

**사례 1**: 산출세액 90만원 (총급여액 3,000만원)
- 90만원 x 55% = 49만 5천원 (공제한도 74만원 내이므로 49만 5천원 공제)

**사례 2**: 산출세액 200만원 (총급여액 6,000만원)
- (130만원 x 55%) + {(200만원 - 130만원) x 30%} = 92만 5천원
- 공제한도: Max[66만원, 74만원 - (6,000만원 - 3,300만원) x 0.008] = Max[66만원, 52.4만원] = 66만원
- 세액공제금액: 66만원 (한도 적용)

## 예외/특이사항
- 외국인근로자가 단일세율 특례(19%)를 신청하는 경우 기본세율이 아닌 19% 적용하며, 비과세/소득공제/세액공제를 적용하지 않음 [p.92]
- 2인 이상으로부터 근로소득을 받는 경우 종된 근무지에서는 기본세율을 적용하여 산출세액을 계산한 후, 주된 근무지에서 합산하여 연말정산 [p.83-84]
